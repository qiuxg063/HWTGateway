> 基于.NET8的跨平台工业物联网网关
>
> B/S架构，可视化配置
>
> 南向连接到你的设备和系统(如PLC、数据库、串口设备、OPC Server、OPC UA Server等)
>
> 北向连接您的物联网平台进行双向数据通讯
>
>

- 内置Mqtt服务(1886,1886/mqtt),支持websocker-mqtt等
- 内置ModbusSlave(模拟设备)，端口503
